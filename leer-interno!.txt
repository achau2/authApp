instalar de manera global nodemon
npm install -g nodemon

instalar localmente en el proyecto 'express' (abrir proyecto con visual studio y en la terminal escribir comando)
npm install express

cambiar en las configuraciones de variables de entorno, a las utilizadas en MongoDB
Entrar a mongoAtlas, levantar la instancia, agregar la IP y luego obtener ruta de conexión y pass.

levantar aplicacion con:
nodemom index.js

npm run dev (llama al script dev que se encuentra en el package.json del proyecto).
